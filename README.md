# coveo-saq-search

> Coveo SAQ Search Coding Challenge

## Build Setup

``` bash
# install dependencies
npm install

# build project for local development
gulp build:local

# build project for production
gulp build:dist

# run unit tests
gulp test
```

## Install

```bash
# get the latest
cd ~/git/coveo-saq-search
git pull

# install using the install script
./install.sh
```
